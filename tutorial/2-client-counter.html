Creating a Client Counter

Now that you know how to get a Gavial template up and running, let us expand the hello world template to a simple client counter.

Objectives

  • Retrieve events from the DOM using UI.listen and event sources
  • Build up state with .fold
  • Create a dynamic HTML interface

Creating and Listening to a Button

First we need a (client) event source, event sources are “open” events on which values can be pushed. A source can be used in combination with UI.listen to create an HTML attribute that connects the DOM to the source as long as it is in use:

import mtfrp.core._
import UI.html.all._

val counterSource = ClientEvent.source[Int]

val ui = ClientDBehavior.constant {
  div(h1("Count!"),
      button("+", UI.listen(onclick, counterSource)(_ => 1)),
      button("-", UI.listen(onclick, counterSource)(_ => -1))
  )
}

In this example the “+” and “-“ button are attached to the onclick event and will forward -1 and +1 resp. to counterSource whenever onclick is triggered.

Building State

Note that ui is still static, let’s change this by first building up our application’s state with .fold:

val state = counterSource.fold(0)(_ + _)

Now state is defined as the sum of all source events starting from 0.

Dynamic UI

Finally, we can use state to build a dynamic interface that shows the current count to the user by wrapping the count in a paragraph tag:

val ui = state.map { c =>
  div(h1("Count!"),
      button("+", UI.listen(onclick, counterSource)(_ => 1)),
      button("-", UI.listen(onclick, counterSource)(_ => -1)),
      p(c)
  )
}

Which brings us to the complete example:

import mtfrp.core._
import UI.html.all._

object HelloApp extends GavialApp {
  val port = 8080
  val host = "localhost"

  val headExtensions = List(script(src := s"hello-fastopt-bundle.js"))

  val counterSource = ClientEvent.source[Int]
  val state = counterSource.fold(0)(_ + _)

  val ui = state.map { c =>
    div(h1("Count!"),
        button("+", UI.listen(onclick, counterSource)(_ => 1)),
        button("-", UI.listen(onclick, counterSource)(_ => -1)),
        p(c)
    )
  }
}
